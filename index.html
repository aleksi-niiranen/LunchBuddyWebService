Lunch Buddy JSON API

This is the API that serves the Lunch Buddy Android application.

Usage

There are two URLs. One for the courses and for rating a course. The URL for courses consists of the restaurant name, year, month and date. URLs look like this http://lunchbuddy.cloudfoundry.com/daily_json/Nutritio/2012/11/14. The response includes a timestamp (date at 00.00:00 in seconds), the name of the restaurant, the courses in all available languages with prices and properties such as gluten free and an id that will be used when rating a course.

{"meta":{"requested_timestamp":1352844000,"ref_title":"Nutritio"},"courses":[{"id":426,"title_fi":"Rapeat kalapalat, tartarkastike","title_en":"Crispy fish pieces","title_se":"Spröda fiskbitar","properties":"L M G","price":"2,20 / 5,20 / 7,20"},{"id":427,"title_fi":"Siskonmakkarakeitto","title_en":"Sausage soup","title_se":"Knackorvsoppa","properties":"G L M","price":"1,75 / 4,00 / 5,50"},{"id":428,"title_fi":"Kikhernepihvit, Jogurttikastike","title_en":"Chickpea patties","title_se":"Kikärtsbiffar","properties":"G L M","price":"2,20 / 5,20 / 7,20"},{"id":429,"title_fi":"Sitruunakiisseli, tallimestarin tapaan","title_en":"Lemon fool","title_se":"Citronkräm","properties":"L M","price":"1,00 / 1,00 / 1,20"},{"id":430,"title_fi":"KAHVIO, Paahtopaistisalaatti","title_en":"Roast beef salad","title_se":"Rostbiffsallad","properties":"G L M","price":"3,50 / 4,00 / 3,50"}]}

Ratings

The API exposes a way to rate a course good or bad. The url for ratings is http://lunchbuddy.cloudfoundry.com/course/rate?id=426&rating=good. The id is included in the JSON response. Value for rating param can be good or bad.
